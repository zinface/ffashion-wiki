# ffashion-wiki

> 访问主页预览地址: http://49.232.192.69:8082/

- 获取本项目源代码
    ```bash
    git clone https://gitee.com/zinface/ffashion-wiki.git
    ```

- 使用 `docsify` 启动

    ```bash
    # 进入已经拉取的项目目录
    cd ffashion-wiki  

    # 启动服务
    docsify serve ./docs
    ```
